# Cranial-Width-Analysis-in-Predynastic-Egypt
Este proyecto analiza si la anchura craneal cambió entre el Periodo Predinástico Temprano y el Predinástico Tardío en Egipto. A través de técnicas básicas de estadística descriptiva, contrastes de normalidad, intervalos de confianza y pruebas de hipótesis, se estudian las diferencias entre ambos grupos.
